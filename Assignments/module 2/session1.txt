(B) List three popular programming languages you have heard of, and for each, mention whether it is usually compiled, interpreted, or can be both.<br><br><em><strong>        	Hint:</strong> Think of languages like Java, Python, and JavaScript.</em>
	
   -> Programming languages such as Java, Python, and JavaScript are widely used for developing software and applications. Java is usually considered both compiled and 	interpreted because it is first compiled into bytecode and then executed by the Java Virtual Machine (JVM). Python is generally an interpreted language, as its code 	is executed by a Python interpreter. JavaScript is also mainly an interpreted language, although modern JavaScript engines use Just-In-Time (JIT) compilation, so in 	practice it can be considered both interpreted and compiled.
